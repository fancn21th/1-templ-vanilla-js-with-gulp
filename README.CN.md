# 原生 JS 脚手架

## 目标

- 一个简单好用的 原生 JS 脚手架
- [KISS](https://en.wikipedia.org/wiki/KISS_principle) 原则
- 易于定制化

## 核心依赖

- gulp@4.0
- gulp-inject
- browser-sync
- del

## 默认的约定

- 源代码目录结构

```
└── src
    ├── module
    │   ├── module.js
    │   └── module.html
    └── app
        ├── main.js
        └── index.html
```

## 安装与使用

- `npm i` 或者 `yarn`

- `npm start`
